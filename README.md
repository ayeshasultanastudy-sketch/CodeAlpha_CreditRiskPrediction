# 💳 Credit Risk Prediction System (Machine Learning Project)

## 📌 Overview

This project is an end-to-end **Machine Learning web application** that predicts whether a loan applicant is a **credit risk or not** based on financial and personal details.

The model is trained using the **German Credit Dataset** and deployed using **Streamlit**, allowing users to input real-time data and receive instant predictions.

---

## 🚀 Features

* Predicts **High Risk / Low Risk** applicants
* Interactive **Streamlit Web App**
* Real-time user input simulation
* Clean and simple UI
* End-to-end ML pipeline implementation

---

## 🧠 Machine Learning Workflow

* Data Cleaning & Preprocessing
* Handling Missing Values
* Feature Encoding
* Model Training (Random Forest Classifier)
* Model Evaluation (Accuracy Score)
* Model Deployment

---

## 📊 Dataset

* German Credit Dataset (CSV format)
* Contains features like:

  * Age
  * Gender
  * Job
  * Housing
  * Credit Amount
  * Loan Duration

## ▶️ How to Run the Project

```

```

### 2. Install Dependencies

```

```

### 3. Train the Model

```
python train_model.py
```

### 4. Run the Web App

```
streamlit run app.py
```
